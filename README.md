# Основные комманды

### pwd
Выводит путь к текущей дериктории

### cd
Помогает менять директорию

### ls
Выводит содержимое дериктории

### touch
Создает файл

### mkdir
Создает директорию

### cp
Копирование файлов

### mv
Перемещение файлов и папок

### cat
Чтение файлов

### rm, rmdir, rm -r
Удаление файлов и папок
